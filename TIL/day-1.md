# study-nestjs

nestjs 학습하기

## 섹션1. 세팅하기

-   nvm 설치 & ...

    ```
    #$ node -v => v22.16.0
    #$ npm -v => 10.9.2

    $ corepack enable
    # 확인: yarn -v => 1.22.22

    $ npm i -g @nestjs/cli
    # 확인: nest -v => 11.0.7
    ```

-   yarn add -> npm install --save로 변경하면 됩니다.

## 섹션2. nest js 맛보기

-   프로젝트 시작하기
    ```
    $ nest new {proj_name}
    ```
-   실행하기
    ```
    $ yarn start:dev
    ```
